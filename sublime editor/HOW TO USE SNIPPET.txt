How to set up:

simply download the file and place it to the sublime user directory.
(you will find the directory in this path:"C:\Users\(username)\AppData\Roaming\Sublime Text 3\Packages\User)"
or,
1.open the file Template.sublime-snippet and copy the codes.
2.copy the codes and open sublime text.
3.Navigate to  Tools>Developers>New Snippet and paste the code.
4.press ctrl+S and save it with anyname.



How to use snippets:
just open then sublime editor and create an new file.
type the snippet trigger name.
